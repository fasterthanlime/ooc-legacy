Name mangling

Name mangling is applied to functions and static members of classes. The followed scheme is the following:

  • Off-class Function

    • Name: my_package__functionName

  • Off-class Variable

    • Name: my_package__variableName

  • Member Variable

    • Struct field name: variableName (no mangling)

  • Static Member Variable

    • Name: my_package_ClassName_functionName_ArgType1_ArgType2_ArgType3

  • Member Function

    • Name: __my_package_ClassName_functionName_ArgType1_ArgType2_ArgType3

    • Struct field name: __functionName_ArgType1_ArgType2_ArgType3

  • Static Member Function

    • Name: see Member Function.

ArgType mangling is the name of the type, appended with "__star" and "__array" as many times as necessary to represent the pointer level of the type.
