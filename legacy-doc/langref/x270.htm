Covers

A cover is a set of functions added to a primitive type. As an example, is here a sample of the lang.String cover (included in the sdk of the standard ooc distribution):

typedef Char* String;

cover String {

  func length -> Int {
    return strlen(this);
  }

}

The definition of String as presented above allows code like:

String word = "anticonstitutionnellement";
printf("the longest french word is %d chars long.\n", word.length);

Covers can't have constructors, thus they can't be instantiated.
