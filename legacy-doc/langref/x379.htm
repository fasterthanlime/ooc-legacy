conditionals

if, else

No surprises:

if(happy) {
  rejoice;
} else if(moody) {
  listenToMusic;
} else {
  cry;
}

switch, case

No surprises either:

switch(action) {
  case DIE: die(); break;
  case MOVE: move(); break;
  case LEAVE: leave(); break;
  default: pray();
}
