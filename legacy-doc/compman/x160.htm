The myth of the lost source files

What happens if you compile a .ooc file without a main function ? Nothing. If the file has no syntax errors, the compiler will parse it, validate it, and happily return.

This is a naive behavior of the ooc compiler. It will be fixed in the next releases, when the possibility of creating easily shared libraries from ooc will be considered more in details.
